Become Self Taught Web developer

If you are ambitious and want to learn coding then learning html first will be best in your interest. So the very first thing you need is your aim to learn web developing and then you need to get an IDE.

What is IDE?

As you can judge from its name IDE(Integrated Development Editor) is editor to edit code files or to write code. There are many IDE’s on internet (Atom, Visual Code, Subline). The editor which I personally use is “Visual Code” and is my favourite of all time. The reason I like it is that it has many extensions which help developers to write code faster than other editors and this editor is light weight which means users with low specifications can use it without any hassle. So lets get your editor ready.
